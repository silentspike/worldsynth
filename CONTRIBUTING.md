# Contributing to WorldSynth

Thank you for your interest in WorldSynth!

## How to Contribute

### Bug Reports

Found a bug? Please use the [Bug Report](https://github.com/silentspike/worldsynth/issues/new?template=bug_report.yml) template to let us know.

Include:
- Steps to reproduce
- Expected vs actual behavior
- System information (distro, audio setup, DAW version)

### Feature Requests

Have an idea? Use the [Feature Request](https://github.com/silentspike/worldsynth/issues/new?template=feature_request.yml) template.

### Security Vulnerabilities

**Do NOT create public issues for security vulnerabilities.**

See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Development

WorldSynth is developed in a private repository. If you are interested in contributing code, please reach out via a feature request issue to discuss collaboration.

### Technology Stack

- **DSP Backend**: Zig 0.14.x
- **UI Frontend**: Svelte 5 + TypeScript + Vite
- **Plugin Format**: CLAP 1.2.7
- **Platform**: Linux (x86_64)

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).
