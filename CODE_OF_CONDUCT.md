# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

We are committed to providing a welcoming and inclusive experience for everyone. We expect all participants to treat each other with respect and professionalism.

## Reporting

Instances of unacceptable behavior may be reported to the project maintainers via GitHub's private messaging or by opening a confidential issue.

## Enforcement

Project maintainers will review and investigate all reports and will respond in a way that they deem appropriate to the circumstances.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
